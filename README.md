Maven dependency
```xml
<dependency>
  <groupId>com.bitplan.pom</groupId>
  <artifactId>com.bitplan.pom</artifactId>
  <version>0.0.8</version>
</dependency>
```

[Current release at repo1.maven.org](http://repo1.maven.org/maven2/com/bitplan/pom/com.bitplan.pom/0.0.8/)

### How to build
```
git clone https://github.com/BITPlan/com.bitplan.pom
cd com.bitplan.pom
mvn install
```
